📌 College Recommendation Chatbot

![image](https://github.com/user-attachments/assets/ae6daa69-b67c-445e-83ef-ed8b08a4e0ad)

This is an AI-powered chatbot that helps users find colleges based on criteria like GPA, tuition fees, location, and ranking. It uses Streamlit for the UI and an LLM (Salesforce T5) to convert natural language queries into structured search filters.


🚀 Features


✅ Upload your own college dataset (CSV format)

✅ Chatbot understands user queries & converts them into structured filters

✅ Uses an LLM to transform queries into Pandas filtering csv

✅ Filters and displays results dynamically
